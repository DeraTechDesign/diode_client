Diode Client

Thank you for installing Diode Client.

Run Client

Open a new terminal and run the following command to get start:

 $ diode --help

Resources

You can find more detail about how diodechain/client works in our previous presentations https://github.com/diodechain/presentations.

You can also find examples in our repository https://github.com/diodechain/diode_go_client#examples.

Uninstall Diode Client

Run the following command to uninstall Diode Client.

 $ sudo bash /Library/Diode/__VERSION__/uninstall.sh
